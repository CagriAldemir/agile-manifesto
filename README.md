## Installation

You can run the project in the development environment or production build. For this, please follow the steps below.

### Development Build

1. Go to root directory of project.

2. Install NPM packages

   ```sh
   yarn
   # or
   npm install
   ```

3. Run Development Build
   ```sh
   yarn start
   # or
   npm run start
   ```

### Production Build

1. Go to root directory of project.

2. Install NPM packages

   ```sh
   yarn
   # or
   npm install
   ```

3. Build Project

   ```sh
   yarn build
   # or
   npm run build
   ```

4. Serve Project
   ```sh
   serve -s build
   ```
